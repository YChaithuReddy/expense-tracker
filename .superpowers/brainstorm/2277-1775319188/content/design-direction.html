Key Upgrades from Reference Designs

1

Grouped Sidebar Nav

Section labels (MAIN, FINANCE, TOOLS, GENERAL) like Kital. Subtle hover states, active indicator bar instead of full background fill. User avatar + org badge at top.

2

Top Header Bar

Global search (Cmd+K style), breadcrumbs, notification bell, user avatar with dropdown — like both references have. Replaces the current bare heading.

3

Rich Stat Cards

Trend indicators (↑12% vs last month), subtle sparkline/progress bars, icon badges. Currently just big numbers + labels — needs visual richness.

4

Premium Tables

Row hover with subtle left-border accent, avatar chips for employees, inline action menus (⋯), sticky headers, pagination. Currently basic with minimal interaction.

5

Refined Spacing & Typography

Larger content padding (40px+), tighter line-heights for headings, Inter font loaded properly, 4px spacing scale. More breathing room like the references.

6

Advanced Components

Empty states with illustrations, skeleton loaders, toast notifications, dropdown menus, modal dialogs with backdrop blur, command palette (Cmd+K).
